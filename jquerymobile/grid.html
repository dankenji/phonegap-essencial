Exemplos jQuery Mobile - Grid

Grid

Grids são 100% largura, completamente invisíveis (sem fronteiras ou fundos) e não têm padding ou margens, para que eles não devem interferir com os estilos de elementos colocados dentro deles. Elementos filho são atribuídos ui-bloco-a / b / c / d / e de uma forma sequencial, que faz com que cada elemento flutue lado a lado, formando a grade.

Grid Simples

Para construir uma de duas colunas (50/50%), utilize uma classe ui-grid-a e adicione dois contêineres filho dentro dela com a classe class="ui-block-a" e class="ui-block-b" respectivamente.
